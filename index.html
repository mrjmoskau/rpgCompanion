Release Notes

I built this rpgCompanion app to help me play Fighting Fantasy games without needing to copy out the adventure sheet, or potentially destroying my book by constant pencil edits.

This is my first crack at the idea, although I have thought about it for a very long time. At one point, it was going to be an HTML page, and then an application for a portable device, and finally, a Java application.

None have come to pass... until now!

There are a few things that are not implemented in ths go-round that I hope I can fix up in a later revision...

  • I want to add a general dice-throwing button, but this version will have to stand without it.
  • There is no support for trying your luck, either during regular play, or during an encounter. The extra dice-throwing feature would have helped here.
  • I based all of the logic on The Warlock of Firetop Mountain, but would like to be able to add on additional rules to support more robust volumes (Creature of Havoc, for example, has an instant-kill rule if you roll double-dice during an encounter).
  • The error checking may not be as robust as it could be, which could lead to some small cheats. Use them wisely, and sparingly to get the full effect of the game.

If you have any old Fighting Fantasy books lying around (who are we kidding, they're in pristine condition up on a book shelf, aren't they?), please try this out, and let me know what you think.
